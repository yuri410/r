This is a simple ray tracer for learning and testing ideas.

## Features
**Materials**
- Lambert Diffuse
- Perfect Mirror
- BRDF (GGX+Schlick)

**Shapes**
- Sphere
- Quad
- Cylinder / disk
- Tri-mesh BVH

**Forward path tracing**
